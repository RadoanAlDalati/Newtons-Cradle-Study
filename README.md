## 📂 هيكلية المشروع (Project Architecture)
تم تنظيم الملفات وفق المعايير الهندسيّة لضمان سهولة التطوير والمتابعة:

| المجلد | الوصف | المسؤول |
| :--- | :--- | :--- |
| `SourceCode/` | الكود المصدري للمحاكاة (Java / Logic Programming) | بلال ورضوان |
| `Documentation/` | التقارير الهندسية، ملفات الـ PDF، والعروض التقديمية | عبد الكريم |
| `Data/` | نتائج التجارب الحقيقية، ملفات Excel، وجداول البيانات | محجوب |
| `Resources/` | الوسائط المتعددة (صور، فيديوهات توضيحية، مخططات) | الفريق |
